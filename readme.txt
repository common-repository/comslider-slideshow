comSlider Plugin allows you to integrate your comSlider slideshows easy, quick and reliable.

== Description ==

comSlider is an online html Slideshow, Gallery or Presentation creator. Fully customizable with amazing visual effects allowing you to build your own slideshow with unique look & feel that just perfect fits with your web page.
So, no coding, no image editing, no learning curve, no software to download. Simply upload your photos, select a template and comSlider will do its magic!

**Main comSlider features**

*stunning slideshow templates, fully customizable
*comSlider "Flying Captions&trade;" animated objects makes your Slider, Gallery or Presentation unique
*fancy transition and visual effects
*online creating, anytime updating and maintenance of your slideshow
*very easy integration into any web page
*no learning curve, no design or coding skills needed, no software to download
*SEO optimized slideshows

**comslider WP Plugin features**

*automatically imports list of your slideshows crated on comslider.com slideshow creator.
*easy integration by selecting a slideshow from the list or by entering comSlider ID and Key.

== Installation ==

1. Upload the plugin to `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the `Plugins` screen.
3. If not already, create your slideshows on https://www.comslider.com (registration is required).
4. Use comSlider Block on the Page Builder to import your slideshows into your WP website.
